## About exercise 

 1.- Create a component that will call an *API, get the data and display each user inside a card. Each user/card next to the other. Display name, username and email in each card. 
 2.- Create an input box at the top of the template that will search by username and will update the template with the list of users/cards matching the searching criteria
 3.- Add a delete button/link inside each card which delete the card and update the list of cards in the template. 

 API https://jsonplaceholder.typicode.com/posts/1/users

 Don't use any css framework, just css. 

 Good luck!
